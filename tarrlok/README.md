# Tarrlok (Laravel app)

This folder contains the **Tarrlok** Laravel application.

For setup, architecture, demo flow, and admin credentials, see the **[project README](../README.md)** at the repository root.

## Quick commands

```bash
composer install
copy .env.example .env
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve
```

App URL: **http://127.0.0.1:8000**
